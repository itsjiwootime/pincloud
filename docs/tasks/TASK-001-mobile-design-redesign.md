## 목표
- LinkMap 모바일 앱의 화면 톤을 하나의 디자인 언어로 통일한다.
- 기능은 유지한 채 `City Field Notes` 컨셉으로 전체 앱 경험을 다시 설계한다.

## 범위
- 공통 테마 토큰 재정의

- 핵심 흐름 화면: 링크 입력, 장소 상세
- 빈 상태, 에러 상태, CTA 위계, 카드/칩/입력창 스타일 정리

## 제외
